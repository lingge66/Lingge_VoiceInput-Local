# Voice Input - 极速本地语音输入

基于 FunASR 的离线语音输入工具，无需联网，毫秒级响应，支持中文标点自动恢复。

## 特性
- **完全离线**：所有模型运行在本地，保护隐私
- **极速识别**：Paraformer-zh 模型 + VAD 静音过滤，识别速度快
- **自动标点**：集成 CT-Punc 模型，输出带标点的通顺文本
- **全局热键**：按 F8 开始说话，松开即识别并粘贴
- **极简 HUD**：32px 高黑色磨砂悬浮窗，不遮挡工作区域
- **托盘控制**：右下角图标显示状态，支持一键退出
